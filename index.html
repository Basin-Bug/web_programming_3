Webプログラミングの演習課題-htmlの要素紹介

概要

このサイトは，日本大学文理学部情報科学科webプログラミングの課題演習2である．主にhtmlの要素について機能を紹介する．

はじめに

このサイトでは，紹介する全ての要素に対して，三つの部分を分ける．まずは概念を紹介し，そしてソースコードを乗せながら毎行つず説明する．さらに，実行結果を表す．

link要素

  1. 概念

    2. link要素では，htmlファイルと外部ファイルの関係を表す．例えばcssやjavascriptなどを使う場合，linkによる構文は不可欠である．

    リンクは空要素であり，そのまま「<, >」を使って始まればよい．

    また，載せられたリンクでは，コンピュータ内の相対パスで乗せられる．サイトと連動するファイルが呼び出させる．

  2. 使用例
    3. 
      /*サイトに関わるものを連動する*/
      < link href="styles/style.css" rel="stylesheet" > /*stylesフォルダ内のstyle.cssを使い，ウェブページを飾る*/
      /*ファビコンを導入する*/
      <link rel="icon" type = "image/png" href="images/naruhodo.png"> /*imagesフォルダ内のnaruhodo.jpgを，ファビコンとして使用する*/
  3. 動作結果

    4. 例えば，CSSではこの行を赤くなるようにすると，こうなる．

    *もちろん，このウェブでの他の部分でもCSSで飾っている．

    また，このサイトのようにファビコンが付けた．

img要素

  1. 概念

    2. img要素では，図を挿入するとき使う．ここで使われる画像形式では，jpg，png，gifなどいろいろのタイプがある．

    img要素では，必須であるsrc属性を持ち，これは画像の相対パスを表す．サイトではこのパスに従って導入する．

    その他，alt要素も使われる．これは必須ではないが，画像を見ることができないユーザーや画像の読み込みが失敗している時，何らかの文字情報を示す．これもかなり便利な機能である.

    CSSを使う場合，そこで画像の大きさ（幅や深さ）を変更することもできる．

  2. 使用例
    3. 
      
      /*読み込まれるケース*/
      <img src="images/onouepanda-t.png" alt="onoue-panda"> /*imagesフォルダ内のonouepanda-tを読み込まれる*/
      /*読み込まれないケース*/
      <img src="images/onouepanda-tttt.png" alt="onoue-panda"> /*同じ操作をしたいが，ファイル名が間違う*/

footer要素

  1. 概念

    2. footer要素が表すものは，一般にサイトのフッターである.主に作者や版権，そしてサイトによっては連絡先がそこに書いている.

    < address>情報で，編集者と連絡先を書かれている.

  2. 使用例
    3. 
      
      <p>picture and favcon by <a href = "https://github.com/onoue-panda" title="onoue-panda">おのうえパンダ制作委員会</a></p> /*このサイトの図についての版権をクレジット表記する*/
      <p> &copy; 5420047 王賛為</p> /*作者情報を表す*/
  3. 動作結果

    4. このペースの一番下のように，フッターがあるようになった.

    また，見やすくなるため，CSSでフッターを青色になるようにした.
